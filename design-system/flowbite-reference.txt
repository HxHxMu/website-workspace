# Flowbite Documentation Summary

Flowbite is "a free and open-source UI component library based on Tailwind CSS" that provides pre-built, interactive HTML components for modern web development.

## Key Features

**Component Library**: Over 56 UI components including buttons, alerts, modals, dropdowns, datepickers, and navbars—all built with Tailwind CSS utility classes and vanilla JavaScript with TypeScript support.

**Design System**: Includes Figma design files, 430+ SVG icons, and a custom ChatGPT model for generating website sections.

**Framework Support**: Works with React, Next.js, Vue, Nuxt, Svelte, Angular, Laravel, Django, Flask, Ruby on Rails, and more through framework-specific libraries or vanilla JavaScript integration.

## Installation Methods

Users can integrate Flowbite via:
- NPM package installation
- CDN links
- Bundled JavaScript import
- Framework-specific packages

## Interactive Components

The library uses two primary approaches for interactivity:

1. **Data Attributes**: "The preferred way to use the interactive UI components from Flowbite is via the data attributes interface which allows us to add functionality via the HTML element attributes."

2. **JavaScript API**: Programmatic control through component classes, supporting both ESM and CJS module formats with TypeScript type declarations.

## Customization

Tailwind v4 moved configuration to native CSS variables in the main stylesheet, allowing developers to customize colors, fonts, spacing, and breakpoints through `@theme` directives.

**Dark Mode**: Enabled via `@custom-variant dark` with optional light/dark mode switching through localStorage and user preferences.

**RTL Support**: Components use logical CSS properties for bidirectional text language support (Arabic, Hebrew, Persian, etc.).
