# Visual Fix Mode

Goal:
Revise the current implementation so it moves closer to the reference screenshot.

Rules:
- prioritize layout and spacing first
- then typography and hierarchy
- then radius, shadow, and color polish
- preserve structure unless structural change is necessary
- do not creatively redesign
- do not copy protected branding assets exactly
- make the smallest effective change set per iteration

Return:
- files to edit
- exact changes
- short rationale
